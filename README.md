# Food Delivery App

### Level Accomplished: **Advanced**
This project fully satisfies all requirements for the **Base**, **Middle**, and **Advanced** levels, including additional "Inspiration" features for a premium user experience.

## Public Access
- **Frontend (Live):** [delivery-app-two-gamma.vercel.app](https://delivery-app-two-gamma.vercel.app/)
- **Backend API:** [delivery-app-server-yulia.onrender.com/api](https://delivery-app-server-yulia.onrender.com/api)

---

## Features Implementation Details

### Base & Middle Levels
- **Full-stack Architecture:** Robust Node.js & Express server with PostgreSQL.
- **Responsive UI:** Fully adaptive mobile-first design built with Tailwind CSS.
- **Dynamic Catalog:** Multi-category filtering and Shop Rating ranges.
- **Smart Sorting:** Real-time sorting by Price (ASC/DESC) and Alphabet (A-Z).
- **Form Validation:** Strict client-side validation for all checkout fields.

### Advanced Level
- **Infinite Scroll:** High-performance batch loading using `Intersection Observer API`.
- **Order History & Search:** Instant search for past orders by Email or Phone number.
- **One-Click Reorder:** Users can repeat any previous order with items instantly added to the cart.
- **Coupon System:** Functional promo codes (e.g., `ELIFTECH`, `HOTDEAL30`) with live discount calculation.

---

## Technical Stack
- **Frontend:** React 19, Vite, Zustand (State Management), Tailwind CSS.
- **Backend:** Node.js (ES Modules), Express.
- **Database:** PostgreSQL (Neon.tech).
- **ORM:** Prisma 7 (with native PostgreSQL adapter for maximum performance).
- **Infrastructure:** Vercel (Frontend), Render (Backend).
